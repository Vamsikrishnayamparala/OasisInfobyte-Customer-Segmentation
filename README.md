# OasisInfobyte-Customer-Segmentation
Customer Segmentation using K-Means Clustering

## Project Overview
This project uses K-Means Clustering to segment customers based on income, spending behavior, and purchase patterns.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Tasks
- Data Cleaning
- Feature Engineering
- Data Standardization
- K-Means Clustering
- Cluster Analysis
- Insights and Recommendations

## Key Findings
- Customers were segmented into four clusters.
- High-income customers showed higher spending patterns.
- Customer groups can be targeted using personalized marketing.

## Files
- Customer_Segmentation_Analysis.ipynb

## Conclusion
Customer segmentation helps businesses identify valuable customer groups and improve marketing effectiveness.
